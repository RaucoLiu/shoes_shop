Shoes_shop

資策會的個人專題（復刻國外的拍賣網站部分功能）

CSS套用了bootstrap，依需求調整並改寫樣式
網頁互動使用了Javascript及JQuery
後端以MySQL資料庫建置，傳接資料則使用PHP
